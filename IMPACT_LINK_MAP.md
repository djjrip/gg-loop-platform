## UTM NAMING RULES

### Standard UTM Format
```
?utm_source=<platform>&utm_medium=<channel>&utm_campaign=<pillar>_<content_type>
```

### UTM Parameters

**utm_source** (Where the link is posted)
- `tiktok` - TikTok posts
- `discord` - Discord server
- `ggloop` - GG LOOP website
- `email` - Email newsletter

**utm_medium** (Type of channel)
- `social` - Social media posts
- `community` - Community platforms (Discord)
- `website` - Website pages
- `newsletter` - Email campaigns

**utm_campaign** (Pillar + content type)
- Format: `<pillar>_<content_type>`
- Examples:
  - `gaming_review` - Gaming setup review
  - `travel_hack` - Travel points hack
  - `sneakers_drop` - Sneaker drop announcement
  - `tools_tutorial` - Creator tools tutorial
  - `culture_story` - Filipino culture story

## PLATFORM-SPECIFIC LINK STRATEGIES

### TikTok
**Link Placement:** Bio link, pinned comment  
**UTM Example:**
```
https://ggloop.io/go/gg-keyboard?utm_source=tiktok&utm_medium=social&utm_campaign=gaming_review
```

**Best Practices:**
- Use short links in bio (character limit)
- Full UTM links in pinned comments
- Update bio link weekly with top-performing offer
- Pin comment immediately after posting

### Discord
**Link Placement:** #deals-and-steals channel, announcements  
**UTM Example:**
```
https://ggloop.io/go/alaska-flights?utm_source=discord&utm_medium=community&utm_campaign=travel_hack
```

**Best Practices:**
- Post in dedicated deals channel
- Use @everyone sparingly (max 1x per week)
- Include context: "Why I recommend this"
- Pin top deals for 7 days

### GG LOOP Website
**Link Placement:** Dedicated affiliate page, blog posts  
**UTM Example:**
```
https://ggloop.io/go/stockx-deal?utm_source=ggloop&utm_medium=website&utm_campaign=sneakers_drop
```

**Best Practices:**
- Create dedicated "Jayson Recommends" page
- Embed in blog posts naturally
- Group by pillar for easy browsing
- Update weekly with fresh deals

### Email (Future)
**Link Placement:** Newsletter, curated lists  
**UTM Example:**
```
https://ggloop.io/go/notion-pro?utm_source=email&utm_medium=newsletter&utm_campaign=tools_tutorial
```

**Best Practices:**
- Feature 1-3 deals per email
- Include personal story/context
- Clear CTA buttons
- Track open rate + click rate

## LINK TESTING CHECKLIST

Before posting ANY affiliate link:

- [ ] Click the link yourself - does it work?
- [ ] Does it redirect to the correct brand page?
- [ ] Are UTM parameters present and correct?
- [ ] Is the short link memorable and on-brand?
- [ ] Does the link match the content pillar?
- [ ] Is the disclosure clear ("This is an affiliate link")?

## TRACKING & ANALYTICS

### What to Track
- **Clicks:** Total clicks per link
- **Conversions:** Total purchases/actions
- **Revenue:** Total commission earned
- **EPC:** Earnings per click (Revenue รท Clicks)
- **CTR:** Click-through rate (Clicks รท Impressions)

### Where to Track
- **Impact.com Dashboard:** Primary source of truth
- **Google Analytics:** Website traffic + behavior
- **TikTok Analytics:** Video views + engagement
- **Discord Analytics:** Message engagement

### When to Track
- **Daily:** Quick check for anomalies
- **Weekly:** Full KPI update (see `IMPACT_KPI_DASHBOARD.md`)
- **Monthly:** Deep dive + strategy adjustment

## TROUBLESHOOTING

**Link not working?**
- Check Impact.com dashboard - is program still active?
- Test link in incognito mode
- Verify UTM parameters are correct
- Contact Impact.com support if broken

**Low clicks?**
- Improve CTA (see `IMPACT_CTA_LIBRARY.md`)
- Better content hook
- Post at better time (TikTok: 6-9pm PST)
- More visible placement (pin comment, bio link)

**High clicks, no conversions?**
- Wrong audience (gaming link to travel audience)
- Bad offer (price too high, not compelling)
- Broken checkout flow (test yourself)
- Cookie window expired (shorten time to purchase)
