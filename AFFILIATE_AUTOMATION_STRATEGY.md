## üöÄ HOW TO ACTIVATE (30 Minutes)

### **Step 1: Run Affiliate Automation (10 min)**

```bash
# Set business email
$env:BUSINESS_EMAIL="jaysonquindao@ggloop.io"

# Run automation
npm run automate:affiliates
```

**What Happens:**
- Opens browser
- Fills Amazon Associates signup form
- Fills G2A Goldmine signup form
- Pauses for you to complete CAPTCHA
- Saves affiliate links

### **Step 2: Get Affiliate Links (10 min)**

**Amazon Associates:**
1. Go to https://affiliate-program.amazon.com
2. Log in with business email
3. Get affiliate links for:
   - Amazon gift cards
   - Gaming gear
   - Electronics

**G2A Goldmine:**
1. Go to https://www.g2a.com/goldmine
2. Log in with business email
3. Get affiliate links for:
   - Steam gift cards
   - Game keys
   - Gaming subscriptions

### **Step 3: Add Links to Rewards (10 min)**

**In Admin Dashboard:**
1. Go to Rewards ‚Üí Edit Reward
2. Add affiliate link to `affiliateUrl` field
3. Set `affiliateCommission` (in cents)
4. Set `affiliateProgram` (e.g., "amazon-associates")

**Example:**
```json
{
  "affiliateUrl": "https://amzn.to/3abc123",
  "affiliateCommission": 250, // $2.50 for $50 gift card
  "affiliateProgram": "amazon-associates"
}
```

### **Step 4: Update Fulfillment Process**

**When User Redeems:**
1. Check if reward has `affiliateUrl`
2. If yes: Use affiliate link to purchase
3. Track commission in database
4. Fulfill reward as normal

**Result:**
- Same fulfillment process
- Automatic commission tracking
- Passive income from every redemption
