6. **Enterprise Partnerships** - Custom pricing

### **The Affiliate Question: DO WE NEED IT?**

**YES. Here's why:**

1. **Passive Income from Existing Users**
   - Every redemption = potential affiliate commission
   - $50 gift card = $2.50-5.00 commission (5-10%)
   - 100 redemptions/month = $250-500/month passive
   - **Zero additional work** - you're already buying gift cards

2. **Scales with Growth**
   - More users = More redemptions = More commissions
   - Doesn't require new features or infrastructure
   - Pure margin improvement

3. **Risk-Free**
   - If affiliate doesn't work, you still fulfill rewards manually
   - No downside, only upside
   - Already built and ready to activate

4. **Competitive Advantage**
   - Lower fulfillment costs = Better rewards = More users
   - Can offer better point-to-dollar ratios
   - More sustainable business model
