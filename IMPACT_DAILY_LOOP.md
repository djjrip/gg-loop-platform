## REVENUE LEVERS (PULL WHEN NEEDED)

### Lever 1: Increase Posting Frequency
**When:** Revenue is growing, engagement is high  
**How:** Post 2x per day instead of 1x  
**Risk:** Audience fatigue, lower engagement per post  
**Test:** Try for 1 week, monitor engagement

### Lever 2: Add New Platform
**When:** TikTok is maxed out (10k+ views per post)  
**How:** Repurpose TikToks for Instagram Reels, YouTube Shorts  
**Risk:** Diluted focus, more work  
**Test:** Start with Instagram Reels, same content

### Lever 3: Integrate into GG LOOP Site
**When:** $500+ revenue per month from TikTok/Discord  
**How:** Create "Deals" section on ggloop.io  
**Risk:** Code changes, CEO approval needed  
**Test:** See IMPACT_ROADMAP_LITE.md Stage 2

### Lever 4: Email Newsletter
**When:** 1,000+ Discord members or 5k+ TikTok followers  
**How:** Weekly "Top Deals" email with affiliate links  
**Risk:** Spam complaints, unsubscribes  
**Test:** Start with bi-weekly, monitor open rate

### Lever 5: Paid Ads
**When:** $1,000+ revenue per month organically  
**How:** Run TikTok ads to top-performing videos  
**Risk:** Money loss if not profitable  
**Test:** $100 test budget, track ROI

## TROUBLESHOOTING

### Problem: No clicks on affiliate links
**Diagnosis:** CTA not clear, link not visible, or audience not interested  
**Fix:**
- Improve CTA (see IMPACT_CTA_LIBRARY.md)
- Pin comment immediately after posting
- Update bio link to top offer
- Test different program

### Problem: Clicks but no conversions
**Diagnosis:** Bad offer, wrong audience, or broken checkout  
**Fix:**
- Test link yourself (click and try to buy)
- Check if offer is competitive (price, value)
- Try different CTA (more context, less pushy)
- Rotate to different program after 2 weeks

### Problem: Views dropping
**Diagnosis:** Algorithm change, content fatigue, or bad hooks  
**Fix:**
- Review hooks (first 3 seconds)
- Test trending sounds/formats
- Post at different times (6-9pm PST)
- Mix affiliate with value content (70/30 split)

### Problem: Revenue plateau
**Diagnosis:** Maxed out current audience, need new levers  
**Fix:**
- Pull Lever 2 (add new platform)
- Pull Lever 3 (integrate into site)
- Test new programs (supporting tier)
- Increase posting frequency (test for 1 week)
