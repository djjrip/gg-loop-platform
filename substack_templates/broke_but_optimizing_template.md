## EXAMPLE POST OUTLINE

**Title:** "How I Flew to Manila for $287 (Budget Travel Breakdown)"

**Hook:**
```
Round-trip flights to Manila cost $800-1,200. For a broke gamer, that's impossible.

I flew for $287. Same airline, same dates, same flight.

Here's the exact strategy I used (and how you can too).
```

**Section 1: The Context**
- Haven't been home to Manila in 3 years
- Family wanted me to visit, but I'm bootstrapping GG Loop
- $1,200 flights = 2 months of server costs
- Had to find a way

**Section 2: The Strategy**
1. **Google Flights Price Alerts** (Free)
   - Set alerts for Manila routes
   - Track prices for 3 months
   - Wait for drops

2. **Flexible Dates** (Save $200-400)
   - Fly Tuesday/Wednesday (not Friday/Sunday)
   - Avoid holidays and peak season
   - Use Google Flights calendar view

3. **Points Stacking** (Save $150-300)
   - Chase Sapphire Preferred (60k points signup)
   - Transfer to airline partners
   - Book through portal for 1.25x value

4. **Budget Airlines** (Save $100-200)
   - Consider layovers (Manila via Tokyo)
   - Pack light (no checked bags)
   - Bring your own food

5. **Book at the Right Time** (Save $50-100)
   - 6-8 weeks before departure
   - Tuesday/Wednesday mornings
   - Incognito mode (avoid price tracking)

**Final Breakdown:**
- Base fare: $287
- Taxes/fees: $73
- Total: $360 (vs. $1,200 normal price)
- Savings: $840

**Section 3: The Cultural Angle**
- My lola taught me: "Marunong magtipid" (know how to save)
- Filipino hustle: work smart, not just hard
- Family first: saved $840 = more money for pasalubong
- Community: share knowledge, help each other

**Section 4: Action Steps**
1. Set up Google Flights alerts today
2. Apply for Chase Sapphire Preferred (60k points)
3. Track prices for 2-3 months
4. Book 6-8 weeks before travel
5. Share your own travel hacks in Discord

**CTA:**
```
That's how I saved $840 on flights to Manila.

If you've got budget travel tips, drop them in the Discord. Let's help each other get home.

discord.gg/ggloop
```

## CULTURAL INTEGRATION GUIDELINES

### Tagalog Usage
- Use sparingly (1-2 words per post)
- Provide translation in parentheses
- Use common words (lola, pasalubong, marunong)
- Don't overdo it

**Good:**
```
"My lola (grandmother) taught me..."
"Bringing pasalubong (gifts) home..."
"Marunong magtipid (know how to save)..."
```

**Bad:**
```
"Kumusta ka na? Matagal na tayong hindi nagkita. Gusto ko pumunta sa Manila pero mahal ang ticket."
(Too much Tagalog, not accessible)
```

- **Family first:** Decisions benefit family
- **Bayanihan:** Community helping community
- **Diskarte:** Hustle, resourcefulness, optimization
- **Utang na loob:** Gratitude, paying it forward
- **Hiya:** Humility, not bragging
