## HOW TO USE THIS QUEUE

This document tracks all outreach messages awaiting CEO approval. Each message has a status:
- **DRAFT** - AG has prepared it, needs CEO review
- **APPROVED** - CEO approved, ready to send
- **SENT** - CEO has sent the message
- **RESPONSE** - Brand has responded

**CEO Workflow:**
1. Review each message in OUTREACH_EMAILS_READY_FOR_APPROVAL.md
2. Mark status as APPROVED if you're ready to send
3. Send the email manually from info@ggloop.io
4. Mark status as SENT with date
5. Track responses
